#################
 Modian Examples
#################

The repository includes a few example projects for various usecases.

``modian-live-example``
   is a minimal example that generates just a live image, documented in
   :doc:`../modian-live-wrapper/project`.

``modian-full-example-*``
   is a complete project based on modian and used to document all its
   features.

   There are variants for all of the currently supported Debian
   releases.

``modian-legacy-example``
   is an example of a modian host system that runs a legacy ISO in qemu.
